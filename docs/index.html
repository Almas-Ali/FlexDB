FlexDB

FlexDB is a simple, fast, and lightweight database for your projects.

It is written in Python and uses Pickle files to store data.

It is easy to use and has a simple API.

It is also open source and free to use.

FlexDB is currently in development.

It is currently in version 0.0.1 alpha.
